Docker Lemonade
===============

Project for runnig Lemonade's projects all together

Preparing to build
---------------
```
git clone git@github.com:eubr-bigsea/docker-lemonade.git
cd docker-lemonade
git submodule init
git submodule update --remote
```

Building and running
---------------

```
docker-compose up
```

Running on detached mode:
```
docker-compose up -d
```
